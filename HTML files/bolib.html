BOLIB: Bilevel Optimization LIBrary of test problems

This first version of the library is made of 124 academic examples of nonlinear bilevel optimization problems, tidied up from a wide range of publications. It provides a uniform basis on which algorithms proposed to deal with nonlinear bilevel optimization can be tested and compared. All the collected examples are programmed via Matlab and the library is freely available from this page; It will later be included in other online platforms.

Download the library

This folder contains all the m-files with the examples (BOLIB > BOLIB-main > BOLIBEexample) and a solver to test them (BOLIB > Solvers).

How to use the library

Details about the library and instructions on how to use it can be found in the following paper. The paper can also be cited when the library is used.

Shenglong Zhou, Alain Zemkoho, and Andrey Tin. BOLIB: Bilevel Optimization LIBrary of test problems. Submitted and available on arxiv, 2018
